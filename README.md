# College-Discovery-Platform
A modern web application designed to simplify the higher education search process. The platform allows students to explore, compare, and discover colleges based on various metrics such as programs, rankings, location, and campus culture, offering advanced filtering and personalized recommendations.

---

## 🚀 Features

- **Advanced Search & Filtering:** Filter colleges by location, streams, course fees, rankings, and facilities.
- **Compare Colleges:** Side-by-side comparison tool to evaluate multiple universities at a glance.
- **Personalized Recommendations:** Get tailored suggestions based on user preferences and academic profiles.
- **Interactive UI/UX:** Clean, responsive dashboard for smooth navigation across devices.
- **Secure Authentication:** User signup/login system to save favorite colleges and personalized shortlists.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **State Management & Tools:** Axios, JSON Web Tokens (JWT)
